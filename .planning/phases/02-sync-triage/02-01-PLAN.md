- Export `syncRequirements(projectId: string)` function
- Read requirements from `.claude/requirements/` directory using fs
- For each `.md` file, insert into Supabase `vibeman_events` table:
  ```typescript

    project_name: projectName,
    payload: {
      requirement_name: filename,
      content: fileContent,
    },
  }
  ```
- Return `{ success: boolean; count: number; error?: string }`
